# DopaMath

**Math is the new Scroll.**

DopaMath is a PWA designed to break the doom-scrolling loop by providing a high-dopamine, productive alternative: solving rapid-fire math problems.

## Documentation
See [docs/](docs/00-vision.md) for full documentation.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build
npm run build

# Lint & Format
npm run lint
npm run format
```
